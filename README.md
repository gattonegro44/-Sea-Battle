⚓ Sea Battle – Морской бой на 7 языках
Классическая игра «Морской бой» в терминале с искусственным интеллектом, разными уровнями сложности и цветным интерфейсом.
Расставляй корабли, стреляй по противнику и топи его флот!
Реализована на 7 языках программирования с единым принципом управления и расширенными возможностями.

🚀 Возможности
Два режима игры – против компьютера (AI) или два игрока (по очереди).

Размещение кораблей – автоматическое или ручное (в некоторых реализациях).

Поле 10×10 – классическая сетка с координатами (A–J, 1–10).

Корабли – 1 палубный (4 шт.), 2 палубных (3 шт.), 3 палубных (2 шт.), 4 палубный (1 шт.).

Искусственный интеллект – простой (случайные выстрелы) или продвинутый (поиск после попадания).

Цветной вывод – свои клетки, клетки противника, попадания, промахи выделены разными цветами.

Статистика – количество выстрелов, попаданий, потопленных кораблей.

Сохранение – опционально (не во всех реализациях).

Подсказки – отображение координат и состояние поля.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [режим] [опции]
Режимы
single – игра против компьютера (по умолчанию).

two – игра для двух игроков (по очереди на одном устройстве).

Опции
-d, --difficulty <easy|medium|hard> – сложность AI (для single).

-p, --place – ручная расстановка кораблей (вместо автоматической).

-s, --save – сохранить состояние игры (опционально).

Управление
Ввод координат в формате A1 или a1 (буква + цифра).

q – выход.

h – помощь (показать управление).

Примеры
bash
# Игра против компьютера (легкий уровень)
python seabattle.py single

# Два игрока
python seabattle.py two

# Средняя сложность с ручной расстановкой
python seabattle.py single -d medium -p

🧠 Логика игры
Поле – 10×10, координаты: столбцы A–J, строки 1–10.

Корабли – размещаются на поле без пересечений и без соприкосновения (по классическим правилам).

Выстрел – игрок вводит координаты, проверяется попадание.

Попадание – клетка отмечается как «ранен/убит», продолжение хода (в классическом варианте).

Промах – ход переходит к противнику.

Победа – когда все корабли противника потоплены.

✨ Дополнительные фичи
Уровни сложности AI:

Easy – случайные выстрелы.

Medium – после попадания стреляет по соседним клеткам.

Hard – использует алгоритм «поиск по шаблону» (сложнее).

Ручная расстановка – игрок может расставить корабли вручную (ввод координат).

Автосохранение – состояние игры сохраняется в файл (в некоторых реализациях).
